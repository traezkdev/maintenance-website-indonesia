Plugin WordPress khusus buatan Indonesia untuk mengaktifkan mode maintenance dengan halaman maintenance yang dapat dikustomisasi dan fitur countdown timer.

== Description ==

Plugin Maintenance Website Indonesia memungkinkan Anda untuk dengan mudah mengaktifkan mode maintenance di website WordPress Anda. Plugin ini menampilkan halaman maintenance yang menarik dan dapat dikustomisasi kepada pengunjung, sementara admin tetap dapat mengakses website secara normal.

Plugin ini dibuat khusus oleh developer Indonesia (TraezKdev) dengan fitur-fitur yang disesuaikan untuk kebutuhan website Indonesia.

= Fitur Utama =

* **Mode Maintenance yang Mudah** - Aktifkan/nonaktifkan dengan satu klik
* **Halaman Maintenance yang Menarik** - Design modern dan responsif
* **Countdown Timer** - Tampilkan waktu perkiraan selesai maintenance
* **Bypass Otomatis untuk Admin** - Admin tetap dapat mengakses website
* **Customisasi Penuh** - Ubah warna, pesan, dan tampilan
* **Preview Mode** - Lihat halaman maintenance sebelum mengaktifkan
* **Auto-disable** - Otomatis nonaktif saat waktu maintenance habis
* **Responsive Design** - Tampil sempurna di semua perangkat
* **SEO Friendly** - HTTP status 503 dan Retry-After header
* **Animasi Menarik** - Floating shapes dan smooth animations
* **Made in Indonesia** - Dibuat dengan cinta oleh developer Indonesia

= Fitur Baru =

* **Ikon Maintenance Kustom** - Pilih ikon yang mewakili status maintenance Anda
* **Teks Progres Dinamis** - Sesuaikan teks yang menampilkan status proses maintenance
* **Teks Kontak Kustom** - Ubah teks informasi kontak sesuai kebutuhan
* **Email Kontak Kustom** - Atur alamat email yang ditampilkan pada halaman maintenance
* **Nomor Telepon Kontak Kustom** - Atur nomor telepon yang dapat dihubungi pengunjung
* **Teks Footer Kustom** - Sesuaikan teks footer yang muncul di halaman maintenance

= Cara Penggunaan =

1. Install dan aktifkan plugin
2. Buka Settings > Maintenance Website
3. Centang "Aktifkan Mode Maintenance"
4. Atur judul, pesan, dan waktu selesai
5. Customisasi warna sesuai brand website
6. Klik "Simpan Pengaturan"

= Fitur Lanjutan =

* **Admin Notice** - Notifikasi di admin saat maintenance aktif
* **Admin Bar Link** - Quick access untuk preview maintenance
* **Dashboard Widget** - Status maintenance di dashboard
* **Auto-save Draft** - Otomatis simpan perubahan
* **Keyboard Shortcuts** - Ctrl+S untuk save, Ctrl+P untuk preview
* **Indonesian Touch** - Special effects untuk Hari Kemerdekaan Indonesia

== Installation ==

= Instalasi Manual =

1. Download plugin zip file
2. Upload ke direktori `/wp-content/plugins/`
3. Extract file zip
4. Aktifkan plugin melalui menu 'Plugins' di WordPress

= Setelah Aktivasi =

1. Buka Settings > Maintenance Website
2. Konfigurasikan pengaturan sesuai kebutuhan
3. Aktifkan mode maintenance

== Frequently Asked Questions ==

= Bagaimana cara mengaktifkan mode maintenance? =

Buka Settings > Maintenance Website, centang "Aktifkan Mode Maintenance", lalu klik "Simpan Pengaturan".

= Apakah admin masih bisa mengakses website saat maintenance? =

Ya, admin yang sudah login tetap dapat mengakses website secara normal, termasuk halaman admin dan frontend.

= Bagaimana cara melihat preview halaman maintenance? =

Klik link "Preview Halaman Maintenance" di halaman pengaturan, atau gunakan link di admin bar saat maintenance aktif.

= Apakah plugin ini mempengaruhi SEO? =

Tidak, plugin ini menggunakan HTTP status code 503 (Service Unavailable) yang memberitahu search engine bahwa website sedang maintenance sementara.

= Bagaimana cara mengatur countdown timer? =

Isi field "Waktu Selesai Maintenance" dengan tanggal dan waktu perkiraan selesai. Countdown timer akan otomatis muncul di halaman maintenance.

= Apakah bisa mengcustomisasi tampilan halaman maintenance? =

Ya, Anda dapat mengubah warna background, warna teks, judul, dan pesan maintenance melalui halaman pengaturan.

= Bagaimana jika lupa menonaktifkan maintenance mode? =

Plugin akan otomatis menonaktifkan maintenance mode jika waktu yang ditentukan sudah habis.
